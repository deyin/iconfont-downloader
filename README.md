# iconfont-downloader

Alimama icon font downloader for android
usage:：
>java -jar IconfontDownloader.jar $id $color [$saveFileName]

$id is the icon file id, which you can search at website: http://iconfont.cn/repositories
$color is the color 16进制格式
$saveFileName is optional argument

e.g:

>java -jar IconfontDownloader.jar 29923 ffffff ic_appreciate.png
